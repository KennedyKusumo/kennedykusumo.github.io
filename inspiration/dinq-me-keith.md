# Style Reference: dinq.me/keith

URL: https://dinq.me/keith

## What Kennedy likes
- The interactive card style — clean bento-box widgets with smooth hover lift + glow
- Career trajectory / timeline card layout
- Overall clean, information-dense feel

## What Kennedy does NOT want
- Network card (circular avatar row showing connections)
- GitHub card/link — does not want to showcase GitHub

## Card visual characteristics observed
- White/light background cards on warm off-white page (#f5f5f5ish)
- Very subtle 1px border at rest (almost invisible)
- Rounded corners ~14px
- Flat at rest — no shadow
- Hover effect: card lifts (translateY ~-4px) + border brightens + subtle shadow appears
- Smooth transition ~200ms
- Icon/logo in top-left of each card
- Each card feels like a self-contained widget with a clear purpose

## Applied to Kennedy's dark site
- Rounded corners 14px
- Hover: translateY(-4px) + accent border glow + elevated shadow
- Cards stay flat (no shadow) at rest

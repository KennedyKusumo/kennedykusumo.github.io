# About Me — Knowledge Base

This folder is where you dump raw material about yourself. Don't worry about
formatting or polish — bullet points, copy-pasted text, brain dumps all work.
I'll read these and turn them into site content.

## Folders

| Folder | What goes here |
|---|---|
| `education/` | Degrees, theses, courses, grades, awards |
| `experiences/` | Jobs, internships, freelance work, volunteering |
| `projects/` | Side projects, research, tools you've built |

## Tips

- One file per thing (e.g. `experiences/syngenta.md`, `projects/msc-thesis.md`)
- Include the messy details — context, constraints, what went wrong, what you're proud of
- Quantified results are gold (%, £, time saved, users, etc.)
- If you have a PDF, slide deck, or document, just drop it in the right folder
